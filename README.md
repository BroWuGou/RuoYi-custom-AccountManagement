## 简介

一直为繁多的账号密码难记而发愁，以前一直记载本子上，感觉不安全也不方便，就开发了一个管理各个站点账号密码的平台
使用了RuoYi的前端框架并做了定制化修改
基本上是在整个RuoYi项目上新增了一个功能
